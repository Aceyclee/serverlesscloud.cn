# 关于 Serverless 中文社区

Serverless 中文社区是一群 Serverless Framework 的开发者们共同发起的，提供关于 Serverless Framework 最新最全面信息、案例和学习资料的网站，致力于推动 serverless 技术在中国的蓬勃发展。
